# Data Structure Notes & Practice

## 📌 專案簡介（Overview）

本專案用於整理**資料結構（Data Structure）相關的手寫筆記**，以及對應的**練習用程式碼**。

內容涵蓋基礎到常見的資料結構，目的在於：

* 加深對資料結構核心概念的理解
* 透過實作練習，熟悉資料結構的操作與應用
* 作為日後複習與查閱的個人筆記庫

---

## 🧠 手寫筆記內容（Handwritten Notes）

手寫筆記主要記錄各種資料結構的：

* 定義與特性
* 操作流程（如插入、刪除、搜尋）
* 時間與空間複雜度
* 使用情境與比較

### 已整理主題：

* Tree
* Heap
* Graph
* Hash Table
* Linked List
* Queue
* Deque
* Stack

---

## 💻 練習程式碼（Practice Code）

本專案包含多個資料結構的練習實作，著重於理解底層邏輯，而非使用內建函式。

### 練習項目：

* Linked List
* List
* Queue
* Tree
* Hash Table

每個資料結構資料夾中皆包含：

* 基本操作實作
* 測試或示範用程式碼
* 說明文件（README 或註解）
